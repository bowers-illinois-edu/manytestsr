# Compute sensitivity bounds for a single block

Compute sensitivity bounds for a single block

## Usage

``` r
compute_sensitivity_bounds(data, outcome, treatment, gamma, test_statistic)
```

## Arguments

- data:

  Block data

- outcome:

  Outcome variable name

- treatment:

  Treatment variable name

- gamma:

  Sensitivity parameter

- test_statistic:

  Type of test statistic

## Value

List with upper and lower p-value bounds
