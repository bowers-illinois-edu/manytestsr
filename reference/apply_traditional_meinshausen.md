# Apply Traditional Meinshausen Procedure

Apply Traditional Meinshausen Procedure

## Usage

``` r
apply_traditional_meinshausen(node_dat, node_tracker, alpha, method)
```

## Arguments

- node_dat:

  Node data

- node_tracker:

  Node tracking structure

- alpha:

  Error rate

- method:

  P-value combination method

## Value

Updated node data with traditional Meinshausen results
