# A set of pre-specified splits using a data.table object (Deprecate)

A set of pre-specified splits using a data.table object (Deprecate)

## Usage

``` r
splitSpecified(bid, x)
```

## Arguments

- bid:

  is not used

- x:

  is a data.table object where each column from 1 to k further divides
  the blocks. Column one should be highest level and each other column
  should be nested
