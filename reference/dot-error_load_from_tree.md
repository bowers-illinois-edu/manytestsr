# Compute error load from an actual tree structure (internal)

Compute error load from an actual tree structure (internal)

## Usage

``` r
.error_load_from_tree(node_dat, delta_hat, z_crit, thealpha)
```

## Arguments

- node_dat:

  data.frame with nodenum, parent, depth, nodesize

- delta_hat:

  Standardized effect size

- z_crit:

  Critical z-value

- thealpha:

  Nominal alpha

## Value

Same structure as compute_error_load
