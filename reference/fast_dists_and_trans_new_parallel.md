# Outcome distances and transformations: C++ OpenMP Parallel version

Outcome distances and transformations: C++ OpenMP Parallel version

## Usage

``` r
fast_dists_and_trans_new_parallel(threads)
```

## Arguments

- threads:

  Is an integer with the number of cores to use.

## Value

A distance creation function taking x (a numeric vector, usually the
outcome) and a variable Z which is not used.
