# Apply Sequential Rejection Principle (Goeman-Solari enhancement)

Apply Sequential Rejection Principle (Goeman-Solari enhancement)

## Usage

``` r
apply_sequential_rejection_meinshausen(node_dat, node_tracker, alpha, method)
```

## Arguments

- node_dat:

  Node data

- node_tracker:

  Node tracking structure

- alpha:

  Error rate

- method:

  P-value combination method

## Value

Updated node data with sequential rejections
