# Example tree-testing data

Synthetic data used in examples and tests for manytestsr.

## Usage

``` r
idt
```

## Format

- idt:

  individual level data.table

- bt1:

  block level data.table

## Source

See `data-raw/make_k_ary_tree_data.R`; produced with TreeTestSim (≥
X.Y.Z) and a fixed RNG seed on 2025-11-08.

## Details

Generated once and stored as package data to avoid slow/regenerative
simulation in examples/tests. The full recipe lives in
`data-raw/make_k_ary_tree_data.R`.

## References

Bowers et al., TreeTestSim, manytestsr.
