# Exact Wilcoxon bound computation (simplified)

Exact Wilcoxon bound computation (simplified)

## Usage

``` r
compute_exact_wilcoxon_bound(w_plus, total_pairs, prob)
```

## Arguments

- w_plus:

  Number of positive pairwise differences

- total_pairs:

  Total number of pairs

- prob:

  Probability parameter

## Value

P-value bound
